#投げ銭コントローラー for PHP
===========

投げ銭BOXに銭を投げるアプリです。  
投げ銭BOXと同じ環境下でapacheなどを走らせ、
ブラウザからアクセスして下さい。
機種を問わず利用する事ができます。


#構成
 * コントローラー[index.php] jQuery/CSS3
 * 送信サーバー[server.php] jQuery/PHP
 * 設定ファイル[config.php] PHP


#動作環境
 * ApacheなどのWebサーバー  
 * コントローラーとBOXが同じローカルIP内にある場合のみ動作  
 * ポートは「8999」を使用します。
 * IPは都度変更して下さい
 * IP、ポートの変更は「config.php」で行います。
